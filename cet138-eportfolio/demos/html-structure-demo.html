Semantic HTML Structure Example

This demo shows how semantic tags organize information for readability, accessibility, and search engines.

Rendered Structure

<header> Site title and navigation links
<main> Primary content container
<article> Main article or standalone topic
<aside> Supporting content or notes
<footer> Author info and copyright

Code Sample

<header>
  <h1>Student ePortfolio</h1>
  <nav>...links...</nav>
</header>

<main>
  <article>
    <h2>What is HTML?</h2>
    <p>HTML structures the web page.</p>
  </article>
  <aside>Quick revision notes</aside>
</main>
