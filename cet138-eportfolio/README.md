# CET138 ePortfolio - Pradip Gaire

## Folder Structure

- `index.html` - Home page
- `pages/about.html` - About Me
- `pages/full-stack.html` - What is Full Stack Development
- `pages/html.html` - HTML
- `pages/css.html` - CSS
- `pages/bootstrap.html` - Bootstrap Framework
- `pages/javascript.html` - JavaScript
- `pages/projects.html` - Projects / Practical Demonstrations
- `pages/contact.html` - Contact Page
- `css/style.css` - Shared custom styling
- `css/demos.css` - Styling for dedicated live demo pages
- `js/main.js` - Shared interactivity and form validation
- `js/demos.js` - Extra logic for full stack and JavaScript live demos
- `demos/full-stack-flow-demo.html` - Full stack flow simulation
- `demos/html-structure-demo.html` - Semantic HTML demonstration
- `demos/css-styled-card-demo.html` - CSS card and box model demonstration
- `demos/bootstrap-grid-demo.html` - Bootstrap responsive grid demonstration
- `demos/javascript-interaction-demo.html` - JavaScript event and validation demonstration
- `images/` - Image assets folder

## Run Locally

1. Open `index.html` directly in a browser.
2. Or run a local server from this folder:
   - `python3 -m http.server 8080`
3. Visit `http://localhost:8080`.
