Section 2: HTML

HTML as the Structure of Every Web Page

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages. It defines content such as headings, paragraphs, images, lists, links, forms, and many other elements.

Without HTML, web browsers would not know how to arrange content. HTML gives meaning to information, making websites readable for users and understandable for search engines.

Basic Webpage Structure

Every HTML document starts with a doctype declaration and includes the <html>, <head>, and <body> sections. The head stores metadata and links to styles/scripts, while the body holds all visible content.

  • <h1> to <h6>: headings
  • <p>: paragraph text
  • <a>: hyperlinks
  • <img>: images
  • <section>, <article>, <footer>: semantic layout

Why Semantic HTML Matters

Semantic HTML uses meaningful tags to describe content clearly. For example, <nav> for navigation and <main> for primary content. This improves accessibility, code readability, and SEO.

Good HTML is not only about showing content; it is about organizing information correctly for humans, browsers, and assistive technologies.

Working Example Code

This sample demonstrates headings, paragraph, link, image, and semantic structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My Simple HTML Demo</title>
  </head>
  <body>
    <header>
      <h1>Welcome to My Portfolio Demo</h1>
    </header>

    <section>
      <p>I am learning HTML structure in CET138.</p>
      <a href="https://www.ismt.edu.np">Visit ISMT College</a>
      <img src="student-photo.jpg" alt="Student profile" />
    </section>

    <footer>Created by Pradip Gaire</footer>
  </body>
</html>

Code Explanation

  • The doctype tells the browser to render in modern HTML5 mode.
  • The <head> contains document metadata and title.
  • The <header> introduces the main page heading.
  • The <section> groups related content about learning.
  • <a> creates navigation to another webpage.
  • <img> uses alt text for accessibility support.
  • The <footer> presents author information.
