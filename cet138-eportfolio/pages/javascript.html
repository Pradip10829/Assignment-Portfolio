Section 5: JavaScript

Adding Interactivity to Web Applications

JavaScript is a programming language used to make websites interactive. It can respond to user actions, update content without reloading the page, validate forms, and control dynamic behavior.

While HTML provides structure and CSS controls appearance, JavaScript creates behavior. This makes web applications more engaging and useful for real users.

DOM Manipulation

The DOM (Document Object Model) represents webpage elements as objects. JavaScript can select these elements and change text, style, classes, and attributes in real time.

Events

Events are user actions such as click, input, submit, and hover. JavaScript listens for these events and runs specific functions when they occur.

Validation & Logic

JavaScript can validate data before submission, show helpful messages, and enforce simple rules. This improves user experience and reduces incorrect input.

Simple JavaScript Example Code

<button id="demoBtn">Click Me</button>
<p id="output">Waiting for click...</p>

<script>
  const button = document.getElementById("demoBtn");
  const output = document.getElementById("output");

  button.addEventListener("click", function () {
    output.textContent = "Button clicked successfully!";
  });
</script>

This example shows event handling with addEventListener() and DOM text updates using textContent.
