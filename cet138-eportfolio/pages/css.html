Section 3: CSS

CSS for Styling, Layout, and Better User Experience

CSS (Cascading Style Sheets) controls how HTML content looks on screen. It handles color, spacing, typography, alignment, and responsive behavior. CSS separates design from structure, making pages easier to maintain and visually professional.

If HTML is the skeleton of a website, CSS is the visual presentation that creates readability, balance, and user-friendly interfaces.

Styling Basics

CSS applies styles using selectors, properties, and values. Example: color, background-color, font-size, padding, and border-radius.

Box Model

Every element behaves like a box: content, padding, border, and margin. Understanding this model helps control spacing and alignment accurately.

Flexbox Basics

Flexbox is a layout system that arranges items in rows or columns with easy alignment control. It is very useful for responsive cards, navigation bars, and centered content.

Styled Card Example (CSS)

.profile-card {
  background: #ffffff;
  border: 1px solid #d7e6f3;
  border-radius: 14px;
  padding: 1rem;
  box-shadow: 0 8px 16px rgba(0, 0, 0, 0.08);
}

.profile-card h3 {
  color: #1f4f72;
  margin-bottom: 0.5rem;
}

.profile-card p {
  color: #4b5e72;
}

.card-wrap {
  display: flex;
  gap: 1rem;
  justify-content: center;
  flex-wrap: wrap;
}

How CSS Improves UI

  • Makes content easier to read through consistent typography.
  • Creates visual hierarchy using spacing and heading styles.
  • Improves navigation clarity with hover effects and layout patterns.
  • Supports responsiveness for mobile and desktop screens.
  • Builds professional identity through color system and visual consistency.
